==== NVIDIA Container Runtime for Docker ====

Bladen boottauksen jälkeen yritän saada Dockerin CUDA-imagea toimimaan, mutta antaa virheen (alla). Asennus on alla, muyös linikit sivuille. JJ/29.1.2019

Linkit:
* https://devblogs.nvidia.com/nvidia-docker-gpu-server-application-deployment-made-easy/
* https://github.com/NVIDIA/nvidia-docker#quick-start
* https://devblogs.nvidia.com/gpu-containers-runtime/
* https://github.com/NVIDIA/nvidia-docker/wiki/Frequently-Asked-Questions#do-you-support-cuda-multi-process-service-aka-mps

Asennus

<pre>
# Add the package repositories
distribution=$(. /etc/os-release;echo $ID$VERSION_ID)
curl -s -L https://nvidia.github.io/nvidia-container-runtime/$distribution/nvidia-container-runtime.repo | \
  sudo tee /etc/yum.repos.d/nvidia-container-runtime.repo

# Install the nvidia runtime hook
sudo yum install -y nvidia-container-runtime-hook
</pre>

Testaus

<pre>
# Test nvidia-smi with the latest official CUDA image
# You can't use `--runtime=nvidia` with this setup.
docker run --rm nvidia/cuda:9.0-base nvidia-smi
</pre>
